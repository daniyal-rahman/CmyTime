**Project Description & Goal**
Build a unified framework that (1) simulates *C. elegans* behavior by driving a soft-body worm model with its full connectome (via c302 + Sibernetic) and (2) trains a parallel RL agent in the same arena. The aim is to compare emergent navigation and obstacle-avoidance strategies between the biologically derived controller and a learned policy.

---

### gym\_wrapper/**init**.py

* Marks the directory as a Python package.
* Ensures both `envs` and `utils` are importable under a single namespace.

---

### gym\_wrapper/envs/worm\_env.py

This file must:

* Define a Gym-style `Env` class for the worm arena.
* Load the NeuroML model and initialize the neural simulator.
* Interface neural outputs (motor-neuron voltages) to muscle activations.
* Step the Sibernetic solver to update body pose and return observations, reward, and done flags.
* Allow an external action to override neural control for RL experiments if provided.

---

### gym\_wrapper/envs/fly\_env.py

This file must mirror `worm_env.py` for a fruit-fly setup:

* Instantiate a Neurokernel-based fly connectome driver.
* Map neural outputs to joint torques in a rigid-body simulator.
* Produce the same Gym API (obs, reward, done) for seamless agent swapping.

---

### gym\_wrapper/utils.py

This file must provide:

* A `NeuralSimulator` class or factory that abstracts over backends (jNeuroML, NEURON, Brian2).
* Methods to (a) inject sensory inputs into the network, (b) advance neural state by one timestep, and (c) read out motor outputs.
* Common helper functions for loading NeuroML files and translating cell voltages into normalized activation signals.

---

### rl\_agent/train.py

This script must:

* Parse `config/default.yaml` for hyperparameters (algorithm choice, learning rate, timesteps).
* Instantiate the appropriate `gym_wrapper` environment (worm or fly).
* Create and train an RL model (e.g. PPO, SAC) on that environment.
* Save the trained policy to `models/`.

---

### rl\_agent/evaluate.py

This script must:

* Load a saved policy from `models/`.
* Run a fixed number of evaluation episodes in the matching environment.
* Compute and log metrics (time to goal, collisions, path efficiency).
* Optionally save trajectory videos or CSV logs for analysis.

---

### models/

* **worm\_full.nml** and **fly\_full.nml**: NeuroML network definitions generated by c302 or Neurokernel.
* **RL model files**: serialized agent checkpoints (e.g. `ppo_worm.zip`).
* Serves purely as an artifacts directory; no code resides here.

---

### notebooks/01\_pipeline\_demo.ipynb

This notebook must walk through:

1. Generating the NeuroML model via a script call.
2. Instantiating the worm Gym environment and stepping it once.
3. Visualizing a snippet of neural voltage traces and body poses.
4. Loading a pre-trained agent and comparing action outputs to neural outputs.

---

### scripts/generate\_model.sh

This script must:

* Accept parameters (e.g. parameter set A/B/C/D, simulation duration).
* Invoke Python commands to call c302’s generator and output `models/worm_full.nml` (and `fly_full.nml`).
* Validate output files exist before returning.

---

### scripts/run\_pipeline.sh

This orchestrator must:

1. Call `generate_model.sh`.
2. Launch the chosen simulation backend (e.g. start Geppetto server if used).
3. Trigger `train.py`.
4. Trigger `evaluate.py`.
5. Summarize outputs (locations of saved models and logs).

---

### config/default.yaml

Contains all adjustable parameters without code changes, including:

* Neural sim settings (dt, duration, parameter set).
* Physics sim settings (timestep, viscosity).
* RL hyperparameters (algorithm, network size, learning rate).
* File paths for NeuroML and model checkpoints.

📋 1. Prerequisites
Hardware

Modern multicore CPU

NVIDIA GPU (for RL training & Sibernetic OpenCL)

OS

Linux (Ubuntu 20.04 LTS recommended) or macOS (Big Sur+)

Software

Git ≥ 2.25

Conda or Miniconda (for environment management)

CMake ≥ 3.15

Java 11+ (for jNeuroML & Geppetto server)

OpenCL drivers (for Sibernetic GPU acceleration)
